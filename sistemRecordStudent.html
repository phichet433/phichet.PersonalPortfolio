System Record Students

Description

Universities where lecturers, staff, or authorities every day need to examine their student records for various purposes such as searching for specific student details and others. In use Examining records manually is a tedious and time-consuming job for a lecturer. Therefore, it is better to develop software within the university that allows such users to be able to insert, update, search or delete records without going through the document manually every time a query arises.
